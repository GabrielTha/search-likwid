 - Gabriel Marczuk Tha

 - Marcus Augusto Ferreira Dudeque



# Setup

    // Clonar o projeto
    git clone git@github.com:GabrielTha/search-likwid.git
    
    // Instalar LikWid
    sudo apt-get install likwid
    
    // Liberar acesso à arquitetura
    sudo modprobe cache
    
    // Compilar o projeto
    make
